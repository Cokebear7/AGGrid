# AGGrid - Car

This is a simple full-stack project which implements AG-Grid for cars. 

### Tech stack

Front-end: JavaScript, React

Back-end: Java, Spring Boot

Database: MongoDB

### Main Features

1. Create new car
2. Update selected car
3. Delete selected car
